calculate_pose(atupleAngVels: tuple, afDeltaT: float, aenumFKMethod: EnumFKMethod = EnumFKMethod.EULER) → Tuple[RobotPose, float, float][source]

calculates the forward kinematics of the mobile robot. atupleAngVels is a tuple of the right and left angular velocities. afDeltaT refers to the sampling time in seconds. aenumFKMethod is the dead-reckoning approach. Default is Euler. Other approaches include: RUNGE_KUTTA and EXACT. Returns the calculated position in the cartesian plane using the class XYPos, Linear and angular velocities at the center of the robot. This should be called everytime the robot changes position.
